步骤 1 / STEP 1: 安装 / Install
─────────────────────────────────────────

macOS/Linux:
  chmod +x setup_portable.sh
  ./setup_portable.sh

Windows:
  Double-click: setup_portable.bat
  双击: setup_portable.bat

⏱️  需要 2-3 分钟 / Takes 2-3 minutes
📦 下载约 200MB 到此文件夹 / Downloads ~200MB to this folder


步骤 2 / STEP 2: 运行 / Run
─────────────────────────────────────────

macOS/Linux:
  ./start_monitor.sh

Windows:
  Double-click: start_monitor.bat
  双击: start_monitor.bat

🎉 GUI 界面会自动打开 / GUI will open automatically
